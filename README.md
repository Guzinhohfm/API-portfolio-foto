# API-portfolio-foto

# Sobre o projeto
- A idéia do projeto foi criar um portfólio/galeria de fotos para que o usuário possa realizar o armazenamento ou até o download de imagens de onde estiver.

# Técnologias utilizadas:

## Front-end:
- React
- JSX
- Bootstrap
- Vite

## Back-end
- ASP.Net Core
- .Net Framework
- C#
